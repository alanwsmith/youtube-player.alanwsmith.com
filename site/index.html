Alan's YouTube Player Web Component

Introduction

This is the web component I'm building to embed YouTube videos on my sites. It's main purpose is to add buttons to control the video and set the playback speed.

Work In Progress

The first version of this component is still under development. You can track the progress in the Features section below.

Current Progress Examples

Features

  • Use a custom <yt-player video="YouTubeId"> tag to do the embed
  • Provide a start attribute to set the starting time
  • Provides a Play/Pause button
  • Provides a Stop button
  • Allow multiple videos on a single page
  • Provide speed adjustment buttons
  • Style via parent page CSS
  • Provide responsive styling example
  • Provide a mute/unmute button
  • Provide a data-state value for styling depending on the state of the player (e.g. playing, paused, stopped)
  • TODO: Provide an autoplay attribute that attempts to auto play the video
  • TODO: Allow other scripts on the page to control the player (this might already work, needs testing)
  • TODO: Show an image instead of the video to start with

Notes

  • I'd like to be able to add a closed caption button but I'm not sure how to do that yet or if it's even possible through the api. TBD on that
  • I'm not trying to optimize loading. The base functionality is provided directly by the YouTube API and default iFrame
