Web Component Styling Issue

It's Fixed!

This page was originally made to troubleshoot an issue I was having with a web component. It's solved now.

The original issue was with the styling of the component. Everything works fine when the component is directly inside the main element of the page. The problem occurred when moving the custom element for the component one level down (e.g. into a section or div tag).

You can see the issue here:

The blue background and red border should surround the video.

The Solution

The fix turned out to be explicitly setting the custom element to display as a block with:

display: block

The main stylesheet for this component has been updated with the fix. (I'm leaving the busted one here as a reference example)

I expect the reason it originally worked in my main element was because of some style applied there. I'm not really worried about it. It's working now.
